# Сёрфер волн — план развития

Мини-апп Telegram. Концепция: считаем **взгляды на тягу, а не воздержание**.
«То, что наблюдает тягу, никогда не курило.»

- Репо: https://github.com/baetovtoxic/serfer-voln
- Прод (GitHub Pages): https://baetovtoxic.github.io/serfer-voln/
- Бот-токен: `8844189542:...` (оставлен прежним по решению пользователя)
- Деплой: правим `index.html` → `git commit` + `git push` → Pages пересобирается ~1 мин.

Делаем по очереди. Отмечаем `[x]` по мере готовности.

## Логика
- [x] **1. Дыхательный пейсер** — eclipseMini делает реальный вдох/выдох (4с/4с) с подписью, вместо декоративного «дыши вместе с ней». (готово 17 июл 2026)
- [x] **3. Инсайты из данных** — в журнале: «волны чаще вечером 19–23», «средняя спадает к 4-й минуте», «пики ниже: было 8 → стало 5». (высокий приоритет)
- [x] **4. Восстановление незакрытой сессии** — активная волна в localStorage; при открытии «волна от 14:32 не закрыта — прошла?». (высокий приоритет)
- [ ] **2. Прогресс волны до 15 мин** — тонкое кольцо у таймера, при пересечении «ты пересидел её пик» + haptic.
- [ ] **5. Пуш-якорь через бота** — cron на VPS: 1 сообщение/день «были волны сегодня?».
- [ ] **6. Экспорт файлом/в чат бота** вместо только буфера обмена.

## Визуал
- [ ] **7. Затмение растёт с прогрессом** — корона ярче/шире на уровнях 10/50/100 волн.
- [ ] **8. Мини-спарклайн кривой** у каждой записи в журнале (curve уже сохраняется).
- [ ] **9. Волна на экране сёрфинга сама успокаивается** со временем, не только от слайдера.
- [ ] **10. Промпт трогать слайдер** раз в 2–3 мин с подсветкой — живые кривые.

## Развитие продукта
- [ ] **11. Универсализация** — не только курение: еда, скролл, алкоголь. Выбор «что серфим» при входе.
- [ ] **12. Онбординг** — 3 карточки на первом входе (что за волна, почему смотрим а не боремся, что считает счётчик).
- [ ] **13. Streak по дням**, не по волнам подряд. Пересмотреть «серия watched подряд» — противоречит идее «без стыда».

## Топ-3 по эффекту
№1 пейсер, №3 инсайты, №4 восстановление сессии.
